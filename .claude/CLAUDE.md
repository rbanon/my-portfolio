# My Portfolio

## Stack
- Angular 17+ (standalone components)
- TypeScript
- Vercel (deploy)

## Estructura
- `src/app/` - componentes principales
- `public/` - assets estáticos

## Comandos frecuentes
- `npm start` - dev server
- `npm run build` - build producción
- `npm test` - tests

## Convenciones
- Componentes standalone
- CSS en archivos separados
- Inglés para código, español para commits

## Lo que NO hacer
- No usar NgModules (proyecto standalone)
- No modificar tsconfig.app.json sin avisar
